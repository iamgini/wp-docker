== Description ==

Grow Your Audience, Enhance your website traffic and engage your visitors with Free to use and Mobile optimized Social Share With Floating Bar WordPress Plugin. You can add/select share icons for  Facebook, Twitter, VK, Google+, Pinterest and Email.

== Installation ==
	= Using The WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Search for Social Share With Floating Bar
	* Click Install Now
	* Activate the plugin on the Plugin dashboard
	= Uploading in WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Navigate to the 'Upload' area
	* Select social-share-with-floating-bar.zip from your computer
	* Click 'Install Now'
	* Activate the plugin in the Plugin dashboard
	= Using FTP =
	* Download social-share-with-floating-bar.zip
	* Extract the social-share-with-floating-bar directory to your computer
	* Upload the social-share-with-floating-bar directory to the /wp-content/plugins/ directory
	* Activate the plugin in the Plugin dashboard
	= Setting Options =
	* Setting Page is located inside Default Settings Option
	* Enable and Disable Post Types Options As Per Need
	= Permalink Setup =
	* Go to Settings -> Permalinks and click on "Save Changes" if your custom post type redirects you to 404 page.
